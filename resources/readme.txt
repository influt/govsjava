Forcepoint Triton Installation Instructions.

Step 1) Extract the Forcepoint Triton_x64.zip to your desktop
Step 2) Right click on the extracted Forcepoint Triton_x64.exe and run as administrator
Step 3) Click continue when prompted with the User Access Control message
Step 4) Follow the Installation Wizard by clicking on the default option
Step 5) Confirm the Forcepoint Triton Icon is visible in the icon tray
Step 6) Delete the Forcepoint Triton_x64.exe from your desktop
Step 7) Reboot
